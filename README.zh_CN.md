# Printemps Web 开发框架

[English][canonical] | [中文-中国][zh-CN]

[__Printemps__](https://www.printempsframework.org/) 是基于 [MoonBit 语言](https://www.moonbitlang.com/) 和 WebAssembly 运行时技术，开发的下一代 Web 开发框架。

注重于在不妥协性能的前提下，同步增强应用程序的安全性、开发体验、可观测性和架构灵活性。
